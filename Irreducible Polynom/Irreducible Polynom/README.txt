Irreducible Polynom

Нахождение неприводимых многочленов над заданным полем простого порядка.
Алгоритм:
	очередной многочлен делим на уже известный неприводимый, меньшей степени.
	Если для любого деления остаток не нулевой, значит многочлен неприводим.

Класс Program.cs точка входа в программу.

Класс PolynomArithmetics.cs реализует арифметику над многочленами(деление).

Класс FinderPolynom.cs осуществляет поиск неприводимых многочленов и запись в файл
	по следующему правилу: имя файла - степень многочленов,
			  в файле: в строке записаны коэффициенты очередного 
			  неприводимого многочлена от младшего разряда к старшему.
Папка с многочленами:.\bin\Debug\Polynomials
